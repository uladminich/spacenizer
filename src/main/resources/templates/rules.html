ОБОЗНАЧЕНИЯ, ИГРОВОЙ ПРОЦЕСС:

# КР - красный ресурс, основной ресурс для выживани (как еда, энергия и т.д.)
# СР - синий ресурс, вспомогательный ресурс (как материалы, топливо и т.д.) при остуствии КР, 2 единицы СР преорбразуются в 1 единицу КР.
# КР(N/N) - влияет на КРАСНЫЙ ресурс первый параметр - увеличивает(если положительный) или уменьшает(отрицательный) добычу на N второй параметр - увеличивает(если положительный) или уменьшает(отрицательный) расход на N
# СР(N/N) - влияет на СИНИЙ ресурс, первый параметр - увеличивает(если положительный) или уменьшает(отрицательный) добычу на N второй параметр - увеличивает(если положительный) или уменьшает(отрицательный) расход на N
# После того, как все игроки сделали ход, раунд заканчивается и происходит добыча ресурсов, затем и расход при необходимости
# Игра заканчивается, если:
- остался только один выживший игрок в начале следующего раунда, он победил.
- не осталось выживших игроков, победятиля нету.
- у всех выживших игроков закончились карты для игры, тогда побеждает тот, у кого больше добытых ресурсов: Сумма КР + половина СР (Если есть карты, влияющие на конвертацию СР в КР, то расчет ковертации идет по их правилам)
# Каждый игрок в свой ход может:
- сыграть карту из своей раздачи
- заменить одну из карт на другую случайную (всего два раза за игру)
- пропустить ход, если у него не осталось карт в руке, а у других игроков они есть
# ГЛОБАЛЬНЫЕ МОДИФИКАТОРЫ - тип планеты, на которой идет игра. Оказывает влияние на всех игроков.
# Есть карты, которые действуют один раунд, потом убираются с поля игроков (см. список ниже)
# Есть карты, которые оказывают эффект единождны, т.е. если на зоне игрока будет больше одной такой карты, то вторая оказывать эффекта не будет (см. список ниже)

КАРТЫ:

- СТАНЦИЯ: КР(+2/+2)
- ДОРОГА: КР(+1/0)
- ШАХТА[КР]: КР(+2/+1)
- ШАХТА[СР]: СР(+2/+1)
- БАР: КР(-1/+1)
- ПЕРЕРАБОТКА ОТХОДОВ: КР(0/-1), СР(0/-1)
- ЛАБОРАТОРИЯ: КР(0/+1), СР(+2/0)
- СОЛНЕЧНЫЕ БАТАРЕИ: КР (0/-1)
- ВЕТРОГЕНЕРАТОРЫ: СР(0/-1)

Активна только одна на поле игрока:
- НАНО ТЕХНОЛОГИИ: КР (+2/0)
- ПЛОХАЯ МЕСТНОСТЬ: КР (-1/+1)
- РОБОТИЗАЦИЯ: Увеличивает показатели карт 'Шахта', 'Дорога', 'Переработка отходов', 'Лаборатория' в два раза.
- КОНВЕРТЕР РЕСУРСОВ: СР (0/+1), Преобразование СР в КР идет 1 к 1.

Активна только один раз в конце раунда, потом убирается с поля:
- ПОМОЩЬ ИЗ ДОМА[РЕСУРСЫ]: Добавляет игроку по 5 КР и СР.
- ПОМОЩЬ ИЗ ДОМА[КАРТА]: Добавляет игроку одну случайную карту.
- ПОЖАР: Игрок теряет по 1 СР и КР.
- ВСПЫШКА БОЛЕЗНИ: Игрок теряет 2 КР.
- ЗАБАСТОВКА РАБОЧИХ: Игрок теряет 3 СР.
- РАЗВЕДКА ОКРУГИ: Добавляет игроку по 1 КР и 2 СР.

ГЛОБАЛЬНЫЕ МОДИФИКАТОРЫ:
- БЕЗ ОСОБЕННОСТЕЙ: не оказывает никаких эффектов.
- ВРАЖДЕБНАЯ ПРИРОДА: КР(-2/0), СР (-2/0)
- ЛЕДЯНАЯ ПЛАНЕТА: КР(0/+1), СР(0/+2)
- ОБШИРНЫЕ ЗАЛЕЖИ РЕСУРСОВ: КР(+1/0), СР(+1/0)
- ПОВЫШЕННАЯ СЕЙСМИЧЕСКАЯ АКТИВНОСТЬ: С некоторой вероятностью в конце раунда может уничтожить до 3 построек (кроме станции) в конце раунда.
